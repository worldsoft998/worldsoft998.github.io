My Modem IP Cores

Chip Design for Satellite Broadband Internet (Satnet)

Mark Chen

Started on 2018-03-31

Integrated Circuits for Mobile, Broadband Satellite Internet Access System without satellite bases, for portable uses for single person, movable on any point of the earth.

The chip will very likely to be developed with fpga technology.

Circuits have been designed and simulated using Mentor Graphics

All circuits were constructed in 90 nm CMOS process using Cadence Virtuoso Suite and were simulated using Spectre.

The technology TSMC 0.18um MMRF CMOS process with a power supply of +/- 0.9v.

DVB-S2X IP integrated into FPGA

IP Core Structure

RS-QPSK Modulator IP Core

The RS-QPSK Modulator IP Core block diagram

The RS-QPSK Modulator consists of a Reed-Solomon encoder and a BPSK/QPSK modulator.

 

RS-QPSK Demodulator IP Core

The RS-QPSK Demodulator IP Core block diagram

The RS-QPSK Demodulator consists of a BPSK/QPSK demodulator and a Reed-Solomon decoder.

IP Core Parameters

RS-QPSK Modem IP Core parameters description

Parameter

Description

W_ADC

ADC Width. Width of the Demodulator input samples from ADC (idati/idatq).

W_DAC

DAC Width. Width of the Modulator output samples to DAC (odati/odatq).

RS(N, K)

Reed-Solomon Codec. Information block length K and coded block length N of Reed-Solomon Codec.

 

Port Parameters Setting

Some input ports that control the IP Core operation need to be set to suit custom configuration.
 

Performance and Resource Utilization
